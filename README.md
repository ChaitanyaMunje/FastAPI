https://d8ekua.deta.dev

GET / Get all courses 
https://d8ekua.deta.dev/courses

POST / Add a new course
https://d8ekua.deta.dev/courses

https://d8ekua.deta.dev/courses/{id}

DELETE / To delete a course with specific id. 
https://d8ekua.deta.dev/courses/{id}

API documentation is present at below link :
